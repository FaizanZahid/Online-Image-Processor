### Because I find matlab boring!

An online image processing tool built using Paintbrush.js.

Allows image editing inside your browser
Supports:

* Noise
* Gaussian Blur
* Zooming
* Greyscale
* Edge detection
* Sepia
* Mosaic
* Color
